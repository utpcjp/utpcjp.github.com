UTPC 2010

東京大学プログラミングコンテスト2010
July 4, 2010

東京大学プログラミングコンテストの第3回目のコンテストである東京大学プログラミングコンテスト2010は2010年7月4日に開催されました．多数の方のご参加をいただき，誠にありがとうございました．

問題セット

5時間のコンテストで以下の12問が出題されました．

  • Problem A: 期末試験!
  • Problem B: 宝くじチェッカー
  • Problem C: コンパイル
  • Problem D: 無矛盾な単位系
  • Problem E: トーマス・ライトの憂鬱
  • Problem F: UTF-8
  • Proglem G: 天体観測
  • Problem H: 迷い猫、走った
  • Problem I: 盗まれた宝石
  • Problem J: 多項式の解の個数
  • Problem K: ワープホール
  • Problem L: 3つのシルエット

また，練習用の問題として下記の4問が出題されました．

  • Problem A: コンパス
  • Problem B: 英作文
  • Problem C: 円周率
  • Problem D: P=NP

問題文・入出力データの利用について

上記の問題文および入出力データは東京大学プログラミングコンテスト実行委員会が作成したものです．これらの問題文とデータの使用・複製・改変等の利用は東京大学プログラミングコンテスト実行委員会の許可なく自由に行っていただいて構いません．ただし，利用によって生じる障害・損害の一切について当方では責任を負わないものとします．
