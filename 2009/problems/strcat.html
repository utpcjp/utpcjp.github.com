Problem B: 文字列連結

2つの文字列 S, T に対して、T のあとに S を結合した文字列を求めよ。

Input

入力の 1 行目、2 行目にそれぞれ文字列 S, T が与えられる。

S, T の長さは、1 文字以上 20 文字以下である。ASCII の印字可能文字のみからなり、空白文字は含まない。

Output

T のあとに S を結合した文字列を 1 行に出力せよ。

Sample Input

009
UTPC2

uck&HaveFun!!
GoodL

S
T

01234567890123456789
01234567890123456789

Output for the Sample Input

UTPC2009
GoodLuck&HaveFun!!
TS
0123456789012345678901234567890123456789

Sample Code in C

#include <stdio.h>

int main(void) {
    char str0[21], str1[21];
    scanf("%s%s", str0, str1);
    printf("%s%s\n", str1, str0);
    return 0;
}

Sample Code in C++

#include <iostream>
#include <string>

using namespace std;

int main() {
    string str0, str1;
    cin >> str0 >> str1;
    cout << str1 << str0 << endl;
    return 0;
}
