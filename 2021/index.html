UTPC 2021

東京大学プログラミングコンテスト2021

東京大学プログラミングコンテスト2021（年度) が2022年3月27日に開催されました。多くの方のご参加ありがとうございました。。

問題セット

5時間のコンテストで全14問が出題されました。

コンテストページ（AtCoder）

  • A : Make UTPC
  • B : Swap and Maximize
  • C : Product Matching
  • D : Chain Graph Pair
  • E : Bounding Box
  • F : Red and Blue Medals
  • G : Matrix Compressor
  • H : Quantum Multiplication
  • I : Card Decks
  • J : Do you like Interval Scheduling Problems?
  • K : Divide Polynomials by #Subset Sums
  • L : Maze Game
  • M : Open the Door
  • N : Tree Swapping
